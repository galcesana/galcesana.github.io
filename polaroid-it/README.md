# Retro Polaroid Creator

A simple browser-based tool for creating a retro Polaroid-style photo from an uploaded image.

## Features

- Drag-and-drop or browse to upload an image
- Auto-crops the image into a square photo area
- Applies a vintage filter and grain effect
- Add a custom caption
- Download the result as a PNG

## Run in browser

This project is a static HTML page, so no server is required.

### Option 1: Open directly

1. Open the folder `polaroid-it`
2. Double-click `index.html`
3. The page will open in your default browser

### Option 2: Drag into browser

1. Open a browser window
2. Drag `index.html` into the browser

### Option 3: Use VS Code Live Server

1. Install the `Live Server` extension in VS Code
2. Open the project folder
3. Right-click `index.html`
4. Choose `Open with Live Server`

## Usage

1. Upload an image using drag and drop or the file browser
2. Enter a caption in the caption field
3. Click `Download Polaroid`

## Notes

- Supports PNG and JPG/JPEG images
- The output is saved as a PNG file
- The project is fully client-side and works offline once loaded
